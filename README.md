# RevPlay Music Streaming Application









\# RevPlay - Music Streaming Application



\## Project Overview

RevPlay is a full-stack monolithic music streaming web application that allows users to discover, play, and manage music. Users can browse songs, create playlists, mark favorites, and view listening history. Artists can upload songs, manage albums, and view analytics about their music.



\---



\## Features



\### User Features

\- User registration and login

\- Browse music library

\- Search songs, artists, albums, and playlists

\- Play songs using integrated music player

\- Create and manage playlists

\- Mark songs as favorites

\- View listening history

\- View profile and account statistics



\### Artist Features

\- Create and manage artist profile

\- Upload songs with audio files

\- Create and manage albums

\- View analytics (play count, favorites, listener statistics)

\## System Architecture

RevPlay follows a monolithic architecture where the frontend, backend, and database are integrated into a single application.



\---



\## Database

The application uses MySQL to store:



\- User information

\- Songs

\- Artists

\- Albums

\- Playlists

\- Listening history

\- Favorites
